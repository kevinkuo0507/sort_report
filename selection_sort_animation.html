🎯 選擇排序法

Selection Sort Animation Demo

一、操作原理

每一輪在未排序區間中尋找最小值，找到後與該輪起始位置交換。左側已完成區會逐步擴大，直到全陣列排序完成。

二、演算法偽代碼

function selectionSort(arr)
  n = length(arr)
  for i = 0 to n - 2
    minIdx = i
    for j = i + 1 to n - 1
      if arr[j] < arr[minIdx]
        minIdx = j
    if minIdx != i
      swap(arr[i], arr[minIdx])

三、時間與空間複雜度

情況複雜度說明
最佳O(n^2)仍需完整掃描找最小值
平均O(n^2)比較次數固定在二次方等級
最差O(n^2)比較次數與資料分布無關
空間O(1)原地排序
