🧬 合併排序法

Merge Sort Animation Demo

一、操作原理

採用分治法。先將陣列切成更小區塊，再把左右兩個已排序子陣列比較後依序寫回，逐步合併成完整排序結果。

二、演算法偽代碼

function mergeSort(arr)
  if length(arr) <= 1
    return arr
  mid = floor(length(arr) / 2)
  left = mergeSort(arr[0..mid-1])
  right = mergeSort(arr[mid..end])
  return merge(left, right)

function merge(left, right)
  result = []
  while left and right not empty
    append smaller one to result
  append remaining elements

三、時間與空間複雜度

情況複雜度說明
最佳O(n log n)分割與合併深度固定
平均O(n log n)每層處理 n 個元素
最差O(n log n)時間複雜度穩定
空間O(n)需要暫存陣列做合併
