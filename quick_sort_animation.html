⚡ 快速排序法

Quick Sort Animation Demo

一、操作原理

選擇一個 pivot，將陣列分成小於等於 pivot 與大於 pivot 的兩側。pivot 定位後，再對左右子區間遞迴做相同分割。

二、演算法偽代碼

function quickSort(arr, left, right)
  if left >= right
    return
  p = partition(arr, left, right)
  quickSort(arr, left, p - 1)
  quickSort(arr, p + 1, right)

function partition(arr, left, right)
  pivot = arr[right]
  i = left - 1
  for j = left to right - 1
    if arr[j] <= pivot
      i = i + 1
      swap(arr[i], arr[j])
  swap(arr[i + 1], arr[right])
  return i + 1

三、時間與空間複雜度

情況複雜度說明
最佳O(n log n)分割相對平均
平均O(n log n)一般資料分布下效率佳
最差O(n^2)每次分割極度不平均
空間O(log n)~O(n)遞迴堆疊深度取決於分割品質
